# Movement Framwork

Movement network được xây dựng trên các robust framework cho phép khởi tạo (creation), triển khải (deployment), và quản lý các rollups dựa trên Move. Các framework này bao gồm:

- MoveStack
- Move Arena
- Move Rollup Framework

```
Mỗi Framework được xây dựng và thiết kế để nâng cao hiệu suất, bảo mật và khả năng tương tác
```
